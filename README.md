# node-yt-downloader
A Node.js app to download Youtube videos with selectable resolution

## Features
- Web interface
- Download videos from YouTube using yt-dlp
- Select from available resolutions
- Outputs MP4 files

## Requirements
- Node.js 14 or higher
- yt_dlp installed on your system `pip install yt-dlp`
- FFmpeg

## Node.js dependencies
- `npm install`
